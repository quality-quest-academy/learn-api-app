# learn-api-app

## English
A simple Node.js API application for learning purposes.

### Getting Started

**Backend:**
```bash
cd backend
npm install
npm start
```
Server runs on `http://localhost:5000`

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
App runs on `http://localhost:5173`

### API Endpoints
- `GET /api/users` - Get all users
- `POST /api/users` - Create user (requires `name` and `role`)
